Install Grunt
=============

    $: npm install -g grunt-cli

Build Swif
==========

Build all in dist/ :

    $: grunt

Build individuals
=================

Compile less stylesheets

    $: grunt less

Minify and Uglify JS

    $: grunt uglify

Copy Files

    $: grunt copy

Build HTML index

    $: grunt targethtml
